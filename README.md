1) Auto-Research-AI
The Auto Research-AI is an autonomous Multi-Agent Research Assistant using techniques like LangGraph and LangChain.

2) The Features of the project are as follows:
- Upload research PDFs
- RAG-based literature review
- Multi-agent workflow using LangGraph
- Paper drafting in LaTeX
- Self-critique and refinement loop
- Web UI using Streamlit

3)Architecture of the project
Planner → Researcher → Writer → Critic → Loop

4)Tech Stack

Python

LangChain

LangGraph

FAISS

OpenAI

Streamlit

5) How to Run

```bash
pip install -r requirements.txt
streamlit run ui/app.py
